Explanation of each point

<AccordionGroup>
  <Accordion title="XAMPP">
    The XAMPP option is only enabled if you are using XAMPP, if you installed MySQL on your own, set "false".
  </Accordion>

  <Accordion title="active_password">
    This is activated if you have a password in the MySQL, otherwise set "false".
  </Accordion>

  <Accordion title="host">
    It is advisable to keep it in "localhost" try not to change this
  </Accordion>

  <Accordion title="user">
    Enter the user where the database you want to back up is located.
  </Accordion>

  <Accordion title="password">
    If you have the "active_password" part activated, you have to enter the password here.
  </Accordion>

  <Accordion title="db">
    Here is the name of the database you want to backup.
  </Accordion>

  <Accordion title="backup">
    If you want to backup a MySQL database have this option enabled, otherwise set "false".
  </Accordion>

  <Accordion title="dump_path">
    If you are not using XAMPP, you have to put the path where MySQL is installed, for example `C:\\Program Files\\MySQL\\MySQL Server 8.3\\bin\\`, remember that you have to put `\\` 
  </Accordion>

</AccordionGroup>
